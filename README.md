# Drive-Cascade Project Page

This repository hosts the GitHub Pages site for the paper:

**Drive-Cascade: Autoregressive Occupancy to LiDAR and Video Synthesis**

## Included content

- Project landing page in [`index.html`](index.html)
- Camera-ready PDF in [`static/pdfs/drive-cascade-camera-ready.pdf`](static/pdfs/drive-cascade-camera-ready.pdf)
- Paper preview image in [`static/images/drive-cascade-preview.png`](static/images/drive-cascade-preview.png)

## Publish with GitHub Pages

If this repository is configured to publish from the default branch root, the page URL is:

`https://summersray.github.io/Drive-Cascade/`

To publish updates:

1. Commit your changes.
2. Push to GitHub.
3. Wait for GitHub Pages to finish deployment.

## Main files to edit later

- [`index.html`](index.html): page content, links, metadata, BibTeX
- [`static/css/index.css`](static/css/index.css): custom styling
- [`static/js/index.js`](static/js/index.js): copy button and small UI behavior

## Notes

- The source PDF remains in the repository root as an original copy.
- The site uses a shorter PDF path under `static/pdfs/` for stable public linking.
